Instruções de uso

    Windows

  • No arquivo recém descompactado, acesse a pasta 'ScanFromWeb' e em seguida o diretório 'dist'. Para rodar a aplicação, apenas execute o arquivo 'ScanFromWeb.exe'.

    • Linux

  • No arquivo recém descompactado, acesse a pasta 'ScanFromWeb' e em seguida o diretório 'ScanFromWeb' dentro de 'dist' . Para rodar a aplicação, apenas execute o arquivo 'ScanFromWeb'. Ou se preferir, execute o script 'run.sh' no diretório raiz.
